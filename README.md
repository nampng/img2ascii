# img2ascii - python

this is just a fun little script that will take an image, convert it to grayscale, and convert the values to ASCII characters.

this results in a nice piece of ASCII art being output to your terminal.

## How it works

Using Pillow, the image will be resized to 100x100 pixels. 
Then, every other row of the image will be read and every pixel converted to it's ASCII equivalent.

The mapping for which characters go to which grayscale value can be found here: http://paulbourke.net/dataformats/asciiart/

## How to use

Place some images in a folder called "img" in the "img2ascii" folder.

`python img2ascii.py [--flip] [{short, long}]`

By default, the scale will be short and unflipped (scale == short, flip == False)

You will then be prompted to choose a file. Enter in the name of the file (with the extension).

The resulting ASCII art will be output to your terminal and a file called output.txt.
